# Читатель Каналов #

* Авторы: Noelia Ruiz Martínez, Mesar Hameed

Это дополнение обеспечивает простой способ читать новости формата Atom или RSS, используя NVDA.
Новостные ленты не будут обновляться автоматически.
Ниже, когда мы упоминаем новостные ленты, мы имеем в виду как RSS, так и ATOM каналы.

## Команды ##

### Диалог Читателя Каналов ###

Доступ к диалогу читателя каналов можно получить из меню nvda, подменю сервиса, пункта каналов.

Он содержит следующие элементы управления:

* Фильтр по: поле редактирования для поиска ранее сохраненных новостных лент.
* Список сохранённых каналов, находящийся в фокусе при открытии диалога.
* Список статей: открывает диалог, в котором представлен список статей из вашего текущего канала. Выберите статью, которую хотите прочитать, и нажмите Enter или откройте веб-страницу выбранной статьи, чтобы открыть соответствующую страницу в браузере. Нажмите кнопку о статье, чтобы открыть диалог с названием и ссылкой выбранной статьи; из этого диалога вы сможете скопировать эту информацию в буфер обмена.
* - Открыть новостную ленту: Открывает выбранную новостную ленту в приложении по умолчанию.
* Открыть канал в формате HTML: открывает выбранный канал в веб-браузере по умолчанию. Вы сможете показать или скрыть даты публикации и кнопки для копирования информации о статьях в буфер обмена.
* Копировать адрес канала: открывает диалог для подтверждения, хотите ли вы скопировать адрес канала в буфер обмена.
* Новый: открывает диалог с полем редактирования, где можно ввести адрес новой ленты. Если адрес действителен и новостная лента может быть сохранена, её имя появится в низу списка новостных лент, исходя из названия новостной ленты.
* Переименовать: открывает диалог с полем редактирования для переименования выбранной новостной ленты.
* Удалить: открывает диалог для удаления выбранной ленты после подтверждения.
* Установить по умолчанию: устанавливает выбранную новостную ленту по умолчанию, и её статьи могут быть доступны с помощью жестов NVDA.
* Импортировать каналы из файла OPML: открывает диалог для добавления новых каналов из файла OPML.
* Сохранить каналы в файл OPML: открывает диалог для сохранения каналов, доступных в диалоге каналов, в файл OPML.
* Параметры: открывает диалог настроек readFeeds, который также доступен в меню NVDA, "Параметры", "Настройки", категория "readFeeds".
* Закрыть: закрывает диалог новостных каналов.

### Примечания #####

* Поле редактирования фильтра можно разместить после кнопки открытия статьи в меню NVDA, "Параметры", "Настройки", категория читателя каналов или после нажатия кнопки настроек в диалоге каналов.
* На этой панели есть возможность отображать даты статей в диалоге списка статей.


### Клавишные команды ###

* Ctrl+Shift+NVDA+Пробел: Объявляет URL текущей статьи. Двойное нажатие откроет веб-страницу.
* Ctrl+Shift+NVDA+8: Обновляет выбранную новостную ленту и объявляет своё последнее название.
* Ctrl+Shift+NVDA+I: Объявляет название и ссылку текущей новостной ленты. Двойное нажатие скопирует название и соответствующую ссылку в буфер обмена.
* Ctrl+Shift+NVDA+U: Объявляет название предыдущей новостной ленты.
* Ctrl+Shift+NVDA+O: Объявляет название следующей новостной ленты.

## Оповещения ##

* Когда название или URL были скопированы.
* Когда не удаётся подключить/обновить ленту, или URL-адрес не соответствует допустимой новостной ленте.
* NVDA отобразит сообщение об ошибке, если новый канал не может быть создан.
* В заголовке диалога списка статей отображается название выбранного канала и количество доступных элементов.
